sectview - Display ELF section layout
=====================================


What
----
sectview displays the section layout of an ELF object file, library, or
executable. 


Building
--------
Merely run `make` from the secview directory.


Running
--------
`./sectview obj`

Where 'obj' is an ELF object file, library, or executable.
